# Event-Management-System

A console-based Event Management System developed in C++ that helps employees and customers manage events efficiently. The project implements file handling, resource management, seating arrangements, venue booking, and event scheduling.

## Project Description

This Event Management System allows users to organize and manage events through a menu-driven interface. The system provides separate functionalities for employees and customers.

Employees can manage activities, events, resources, and users, while customers can book venues, reserve seats, choose food and vendors, and check event availability.

## How to Run

Clone the repository:

```bash
git clone https://github.com/AaizaAamer/Event-Management-System-CPP.git
```

Open the project in:

- CodeBlocks
- Dev C++
- Visual Studio
- VS Code

Compile and run:

```bash
g++ main.cpp -o eventsystem
./eventsystem
```
